## 📦 Instalación
1. Descarga el archivo `.zip` desde el repositorio.  
2. Descomprime el contenido en tu carpeta de preferencia.  
3. Configura las variables necesarias en el archivo de configuración.  
4. ¡Listo! Ya puedes ejecutar el proyecto.
